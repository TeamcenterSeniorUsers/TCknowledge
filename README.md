## 知识库 MarkDown 分级规范

1. 根据内容考虑用" #,##,### "依次表示高级别主干分级
2. 次级别分支用" -,*,+ "三种符号区分
3. 所有“-”的优先级高于“*”
4. 所有“*”的优先级高于“+”
